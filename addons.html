Installing Addons¶

  • make the addons code available to Zope for import

vagrant ssh and change the buildout.cfg file in /vagrant/training.

Add eggs to [instance]:

  • Products.PloneFormGen
  • Products.LinguaPlone
  • collective.plonetruegallery
  • plone.app.themeeditor
$ bin/buildout
$ bin/instance fg

code is importable got registered via ZCML.

  • /@@plone_control_panel > Addons
  • install all four packages

PloneFormGen¶

Creating forms in Plone:

  • pure: html and python in a view
  • framework: z3c.form, formlib, deform
  • ttw: Products.PloneFormGen

Registration-form:

  • Add ‘Form Folder’ called ‘Registration’
  • QuickEdit
  • Remove field “Subject”
  • Add fields for food-preference and shirt-size
  • Add DataSave Adapter
